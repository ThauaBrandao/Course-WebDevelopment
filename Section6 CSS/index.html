CSS Inline

CSS Inline serve para aplicar o estilo na própria linha/na tag que eu quero formatar

Com o código style="color: red;"

(não é muito recomendado)

Seletores

Também dá para aplicar o CSS na própria página do HTML

Basta criar um style no head

(não é muito recomendado)

CSS Externo e Classes/ID

Para usar o CSS Externo basta criar um style.css na pasta do projeto e usar os comandos lá

Usar . para Classes

Usar # para ID
